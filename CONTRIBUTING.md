# Contributing

lenslab is a solo, early-stage project. It is **open source but not open collaboration**: external
contributions — issues, pull requests, feature requests — are **not being accepted** at this time.

This may change once the project reaches a stable release.

**Security vulnerabilities** are the exception — those are always welcome. See
[`SECURITY.md`](SECURITY.md) for private reporting instructions; do not file them in public.

## Design

The design is the source of truth. Before reasoning about how the code is shaped, read
[`docs/GENESIS.md`](docs/GENESIS.md) and [`docs/DECISIONS.md`](docs/DECISIONS.md), then the rest of
[`docs/`](docs/). Execution status lives in [`docs/ROADMAP.md`](docs/ROADMAP.md).
